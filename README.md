# 🖼️ CIFAR-10 Image Classification — CMPUT 328

Image classification on the CIFAR-10 dataset built as part of CMPUT 328
(Visual Recognition) at the University of Alberta. CIFAR-10 consists of
60,000 32×32 colour images across 10 classes including planes, cars, birds,
cats, and more.

## 🛠️ Built With

- Python
- PyTorch
- Jupyter Notebook

## 📁 Structure

```
└── CIFAR.ipynb    # Model training, evaluation, and results on CIFAR-10
```

## 🚀 Getting Started

1. Clone the repo
2. Install dependencies: `pip install torch torchvision numpy matplotlib`
3. Launch the notebook: `jupyter notebook CIFAR.ipynb`

> The CIFAR-10 dataset will be automatically downloaded via `torchvision.datasets.CIFAR10`.
